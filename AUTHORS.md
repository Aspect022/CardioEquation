## 🤝 How to Be Listed

Want to be recognized in this file? Here's how:

1. **Code Contributors**: Make a merged pull request
2. **Bug Reporters**: Report a valid bug that gets fixed
3. **Documentation**: Improve documentation with a merged PR
4. **Research**: Cite CardioEquation in your academic work
5. **Community**: Active participation in discussions and helping others

## 📝 Attribution Guidelines

When adding contributors:
- Include their preferred name or username
- Link to their GitHub profile (if available)
- Describe their contribution
- Get their permission before adding personal information

**Example:**
```markdown
- **[Name](https://github.com/username)** - Brief description of contribution
```
